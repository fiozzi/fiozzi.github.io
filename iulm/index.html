  • Assignment Finale: Vlookup in python
    Descrizione: Scrivere una funzione che svolge lo stesso compito svolto dalla funzione VLOOKUP di Excel (in italiano è la funzione CERCA.VERT).
    La funzione accetta quattro parametri: nomefile: il nome di un file csv da analizzare
    campo: la stringa del campo in cui si vuole cercare un certo valore
    valore: il valore di quel campo
    output: il campo da restituire
    e restituisce una lista degli OUTPUT delle righe che verificano la condizione CAMPO = VALORE.

    Ad esempio, si consideri il file bridges.data che abbiamo utilizzato nelle lezioni precedenti:
